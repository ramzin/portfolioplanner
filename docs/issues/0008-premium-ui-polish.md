## What to build

Perform a comprehensive review and implementation of premium frontend aesthetics. Revamp the styling to use custom typography, dark-mode glassmorphism panels, CSS variables, interactive micro-animations, and input masks formatting large numbers cleanly with thousands separators.

## Acceptance criteria

- [ ] Typography uses a modern Sans-Serif font family (e.g. Google Fonts Outfit/Inter) instead of default browser fallbacks.
- [ ] UI layout uses dark-mode glassmorphism cards (blurred background, subtle semi-transparent borders, and elegant drop shadows).
- [ ] Number inputs format 7-digit numbers (like €1,500,000) with thousands separators during input or on blur, avoiding scientific notation.
- [ ] Sliders and buttons have interactive hover and active states with CSS transitions (e.g. scale, color change, glow effects).
- [ ] The dashboard layout is fully responsive, looking stunning on desktop and tablet viewports.
- [ ] Perform a design-review session with the user to refine colors, layout spacing, and charting options.
